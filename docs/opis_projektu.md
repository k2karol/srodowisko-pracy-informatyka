# Opis Projektu

## Zarządzanie Warsztatem Samochodowym

Projekt dotyczy aplikacji pomagającej zarządzać warsztatem samochodowym. 

### Główne zadania
- Szybki podział obowiązków między pracowników
- Sprawdzenie listy ich obecności
- Sprawdzanie i zarządzanie listą części na magazynie
- Zarządzanie zleceniami do wykonania

### Odbiorcy
- Właściciele warsztatów samochodowych
- Pracownicy warsztatów samochodowych
