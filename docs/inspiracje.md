

### Frontend
- Dashboard (ekran główny)
- Moduł logowania
- Formularz przydzielania zadań
- Lista obecności
- Magazyn (przegląd i zarządzanie)
- Lista zleceń
- Raporty i statystyki
- Panel zarządzania uprawnieniami

### Backend
- API REST do komunikacji z frontendem
- Serwer bazy danych
- Logika biznesowa dla każdego modułu
- System uwierzytelniania (JWT/sesje)
- System powiadomień

### Baza danych
- Tabela pracowników
- Tabela zadań
- Tabela obecności
- Tabela części zamiennych
- Tabela zleceń
- Tabela historii/logów
- Tabela uprawnień użytkowników

### Funkcjonalności
- Przydział zadań mechanikom
- Śledzenie wykonania zadań
- Rejestracja czasu pracy (obecność)
- Zarządzanie magazynem (stan, alertów)
- Obsługa zleceń serwisowych
- Generowanie raportów
- System powiadomień
- Zarządzanie rolami i uprawnieniami

### Technologia (propozycja)
- **Backend:** Node.js/Python/Java
- **Frontend:** React/Vue/Angular
- **Baza danych:** PostgreSQL/MySQL
- **API:** REST/GraphQL
- **Autentykacja:** JWT

## Wstępne dane wejściowe

- **Logowanie** - informacje o użytkownikach i ich rolach
- **Zadania** - typy prac, statusy, priorytety
- **Obecność** - rejestr pracowników
- **Magazyn** - katalog części z kategoriami i stanem
- **Zlecenia** - dane pojazdu, klienta i naprawy
- **Uprawnienia** - role dostępu
- **Raporty** - dostępne analizy i metryki

## Źródła pomysłów

- Doświadczenie w pracy w warsztacie, chęć uproszczenia i przyśpieszenia ich funkcjonowania.
